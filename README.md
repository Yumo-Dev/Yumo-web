# Welcome to Yumo web!

YUMO application with interactive features including games, community, and task management.

## Tech Stack

- Vue 3 + Vite
- Web3.js & Ethers.js
- Element Plus & Vant UI
- Vue Router & Vuex
- ECharts
- i18n (Multi-language support)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
